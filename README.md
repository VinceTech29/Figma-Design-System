# Figma Design System – UI/UX Layout

## Overview
This project is a Figma-based UI/UX layout focused on building a structured and scalable design system. It includes brand framing, color palettes, typography, color variants, and system color guidelines.

## Sections Included

### 1. Frame Sample
- Brand name placeholder
- Layout structure preview
- Basic visual composition reference

### 2. Color Palette
- Primary color
- Secondary color
- Accent color
- Color psychology notes
- Usage context for each main color

### 3. Typography
- Primary font family
- Secondary font family
- Heading styles (H1–H6)
- Body text styles
- Caption and small text
- Font weights and hierarchy
- Line height and spacing guidelines
- Consistent text scaling system

### 4. Color Variant
- Primary color scale (light to dark shades)
- Secondary color scale
- Accent color scale
- Organized for UI hierarchy and accessibility

### 5. System Color
- Success
- Error
- Warning
- Info
- Functional color mapping for interface feedback

### 6. Color Variant v2
- Extended red scale
- Extended blue scale
- Extended yellow scale
- Additional grayscale variations

### 7. System Color v2
- Refined semantic color system
- Clear state indication structure
- UI consistency across components

## Purpose
- Establish a consistent visual identity
- Improve UI scalability
- Maintain color harmony and accessibility
- Provide structured guidelines for developers

## Tools Used
- Figma
